How Flexbox Works

Flexbox is a newer method of layout that allows for flexible positioning of elements. To start this the container holding all elements must have the display:flex property and value in the css

Flex layout is one dimensional in that elements are positionedin either horizontally using the "flex-direction" property. The flex-direction value can be either row OR column. However, the default direction is always row, so flex will flex horizontally unless told to flex vertically using the column value. the CSS property for this is flex-direction.

The flex-wrap: wrap specifies that the flex items will wrap if necessary:

If the direction is row, elements can be positioned horizontally by using the justify-content property.

There are several positions: start is the default and will ways start on the left, center, then end will position to start on the right. "space-around" will add equal space around each item. "space-between' will add equal space in-between each item.

If the direction is row, elements can be positioned vertically by using the align-items property.

There are several positions: start is the default and will ways start on the top, center, then end will position to start on the bottom. "space-around" will add equal space around each item. "space-between' will add equal space in-between each item. The "stretch " value of the align-items property fill stretch the element from top to bottom. This is the default so this will happen automatically if there is no align-items listed.
